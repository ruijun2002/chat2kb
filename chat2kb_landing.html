谁在用 Chat2KB

💬 场景 1：客户技术咨询
"MySQL 和 PostgreSQL 的高可用方案有什么区别？"
→ 结构化对比表 + 架构图 + 配置建议，可直接转发客户
🧪 场景 2：POC 支持
"客户要测 Redis 集群性能，推荐什么配置？"
→ 根据客户业务规模，推荐实例规格 + 预期性能指标
🎯 场景 1：高意向客户发现
"本周高意向客户 TOP 5？"
→ 意向评分排序 + 关键行为摘要 + 建议跟进时间
📊 场景 2：客户试用深度评估
"客户 A 的试用深度如何？值得推进吗？"
→ 试用功能清单 + 使用频率 + 团队参与度 + 转化建议
⚠️ 场景 1：沉默客户预警
"30 天未登录的客户有哪些？"
→ 沉默客户列表 + 最后活跃时间 + 建议唤醒策略
🔍 场景 2：客户体验卡点
"客户操作失败集中点在哪里？"
→ 错误类型分布 + 影响客户数 + 优先级修复建议
📈 场景 1：资源效率评估
"三个环境资源利用率对比？"
→ 多环境对比图表 + 低效集群清单 + 优化建议
💰 场景 2：成本优化
"资源浪费 Top 10 集群是哪些？"
→ 资源利用率排名 + 预估节省成本 + 回收建议
📊 场景 1：功能渗透分析
"自动扩缩容功能的使用率是多少？"
→ 功能使用分布 + 趋势变化 + 推广建议
🔄 场景 2：Onboarding 漏斗
"客户 onboarding 漏斗各步转化率？"
→ 注册→创建集群→配置→活跃 各步转化率 + 流失点分析
📚 场景 1：技术文档速查
"怎么配置备份策略？"
→ 步骤指南 + 代码示例 + 注意事项 + 相关文档链接
